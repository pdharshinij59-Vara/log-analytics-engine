# Log Analytics Engine

A simple backend analytics system that processes application logs and extracts useful insights such as error rates and user activity.

## Features

- Detect application errors
- Analyze user activity
- Process log data
- Simple analytics using Python

## Tech Stack

- Python
- Data Processing
- Backend Analytics
